Method

Enrollment: The app uses the Urban Institute CCD Directory API to find districts and the CCD Enrollment API grade-level endpoints to retrieve K–12 enrollment. It requests each CCD year from 2015 through 2024 for the selected LEAID, corresponding to school years 2015–2016 through 2024–2025.

Forecast: The forecast applies the average annual percent change from the three most recent valid year-to-year changes to the latest actual enrollment, then compounds that average rate forward five years. It is intended as a quick planning signal, not an official projection.

Comparisons: Peer districts are selected within the same state by nearest latest-year enrollment. The chart compares percent change from the first available year to the latest available year.
