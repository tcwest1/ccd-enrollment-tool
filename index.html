Method

Enrollment: The app uses CCD district enrollment/membership endpoints from the Urban Institute Education Data Portal. It requests each year from 2015 through 2024 for the selected LEAID.

Forecast: The default forecast is a simple ordinary least squares linear trend using the ten available annual points. It is intended as a quick planning signal, not an official projection.

Comparisons: Peer districts are selected within the same state by nearest latest-year enrollment. The chart compares 10-year percent change.
