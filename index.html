Method

Local files: The app expects school directory files by school year and state in ccd-enrollment-tools/json/directory/, such as ccd-enrollment-tools/json/directory/ccd_sch_directory_2425_NM.json, and school membership files by school year and state in ccd-enrollment-tools/json/membership/, such as ccd-enrollment-tools/json/membership/ccd_sch_membership_2425_NM.json.

Enrollment: After a district is selected, the app loads the local CCD school membership file for each school year/state, filters to the selected LEAID, and sums K–12 membership.

Aggregation: The app aggregates school-level membership rows to the selected LEAID and sums grades K–12 only. Pre-K, grade 13, ungraded, and adult education are excluded when grade fields are available.

Forecast: The dotted projection line begins with the 2018–2019 school year. For each projected point, the tool applies a rolling three-year average annual percent change to the prior year's enrollment, then compounds that same rolling method forward five years beyond the latest actual year. It is intended as a quick planning signal, not an official projection.

Number of Schools: The table counts unique schools with K–12 membership in each loaded school membership file. If membership rows are unavailable, it falls back to the matching school directory file when available.
