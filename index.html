Method

Local files: The app expects state-level LEA directory files in the main repository folder, such as ccd_lea_directory_nm.json, for state and district lookup.

Enrollment: After a district is selected, the app calls the Urban Institute Education Data Portal CCD district enrollment API for each school year and sums grade-level membership for kindergarten through grade 12. It tries both padded and unpadded LEAID formats when needed.

Aggregation: The app sums the API’s grade-level records for K–12 only. Pre-K, grade 13, ungraded, and adult education are excluded. A school year is treated as incomplete unless all 13 K–12 grade groups return data: kindergarten and grades 1–12. Each grade group can try multiple Urban endpoint aliases, such as grade-kg, grade-k, and grade-kindergarten for kindergarten.

Forecast: The dotted projection line begins with the 2018–2019 school year. For each projected point, the tool applies a rolling three-year average annual percent change to the prior year's enrollment, then compounds that same rolling method forward five years beyond the latest actual year. It is intended as a quick planning signal, not an official projection.

Number of Schools: The table calls the Urban Institute CCD district directory API by LEAID and school year to retrieve the annual operational school count when available.
